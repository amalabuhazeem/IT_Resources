Welcome to IT Resources

This page will help you understand how to use our website effectively.

1. Home Page

The home page provides the latest updates and features. Navigate to other sections using the menu at the top. Admins can access designated pages via their icon, while library users can proceed to the materials page to perform their tasks.

2. Material Page

This page allows users to choose and search for materials. Users can also communicate with officials for guidance and submit feedback or report issues. To exit, press the Close button.

3. Any Material Page

Within any subject, students will find the prescribed material, summaries, and related questions. Additional resources and a discussion group for the subject are available, along with a comments box and evaluation feature for the library.

4. Prescribed Material and Annexes

Students will find all necessary course material, summaries, and questions divided by unit, along with additional features as previously mentioned.

5. Resources Page

This page provides additional sources of information to expand understanding and enhance academic performance.

6. Last Page

The final page thanks students for using the library and invites them to leave feedback to help us improve.
